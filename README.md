# Zerg

本项目基于 ThinkPHP 6.0 构建 RESTFul API。

> 运行环境要求PHP7.2+，兼容PHP8.1

## ThinkPHP

- 三大核心：路由、控制器、模型
- 验证器、读取器、缓存与全局异常处理
- ORM：模型与关联模型

ThinkPHP 6.0 开发手册：[https://www.kancloud.cn/manual/thinkphp6_0](https://www.kancloud.cn/manual/thinkphp6_0)
